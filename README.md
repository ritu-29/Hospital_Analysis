
📊 Hospital Data Analysis Project – 

🏥 Project Title

Comprehensive Hospital Data Analysis using Microsoft Excel

⸻

📌 Introduction

In today’s data-driven world, healthcare institutions generate a large amount of data related to patients, treatments, doctors, and operational costs. Proper analysis of this data helps hospitals improve efficiency, reduce costs, and enhance patient care.

This project focuses on analyzing hospital data using Microsoft Excel. It demonstrates how Excel tools like Pivot Tables, charts, and formulas can be used to extract meaningful insights from raw data.

⸻

🎯 Objective

The main objectives of this project are:
	•	To organize and analyze hospital data efficiently
	•	To identify patterns in patient admissions and treatments
	•	To evaluate hospital department performance
	•	To analyze treatment costs and financial data
	•	To create an interactive dashboard for easy visualization
	•	To improve decision-making using data insights

⸻

📂 Dataset Description

The dataset used in this project contains structured hospital-related information. Below are the details of each column:
	•	Patient ID – Unique identifier for each patient
	•	Patient Name – Name of the patient
	•	Age – Age of the patient
	•	Gender – Male/Female
	•	Disease – Type of illness diagnosed
	•	Admission Date – Date when patient was admitted
	•	Discharge Date – Date when patient was discharged
	•	Doctor Name – Assigned doctor
	•	Department – Hospital department (e.g., Cardiology, Neurology)
	•	Treatment Cost – Total cost incurred for treatment

⸻

🛠 Tools and Techniques Used

This project is completed using the following Excel tools:

1. Microsoft Excel

Used as the primary platform for data storage, cleaning, and analysis.

2. Pivot Tables
	•	Summarize large datasets
	•	Analyze patient count, cost, and department performance

3. Charts and Graphs
	•	Bar Charts → Department comparison
	•	Pie Charts → Gender distribution
	•	Line Charts → Trend analysis

4. Slicers
	•	Used to filter dashboard interactively

5. Excel Functions
	•	SUM() → To calculate total treatment cost
	•	AVERAGE() → To find average cost
	•	COUNT() → To count number of patients
	•	IF() → For conditional calculations
	•	VLOOKUP() → To fetch data from tables
	•	DATEDIF() → To calculate hospital stay duration

⸻

🔄 Data Cleaning Process

Before analysis, the data was cleaned using the following steps:
	•	Removed duplicate records
	•	Checked for missing values
	•	Standardized date formats
	•	Corrected spelling errors in names and departments
	•	Ensured consistent data types

📊 Data Analysis Performed

1. Patient Analysis
	•	Total number of patients admitted
	•	Age group distribution
	•	Gender-based patient analysis

2. Department-wise Analysis
	•	Number of patients per department
	•	Department workload comparison

3. Disease Analysis
	•	Most common diseases
	•	Frequency of each disease

4. Cost Analysis
	•	Total treatment cost
	•	Average treatment cost
	•	Cost distribution across departments

5. Doctor Performance Analysis
	•	Number of patients handled by each doctor
	•	Comparison of doctor workloads

6. Time Analysis
	•	Average hospital stay duration
	•	Admission trends over time
